## Setup

1. Make sure to install the dependencies:

```bash
# yarn
yarn install

# npm
npm install

# pnpm
pnpm install --shamefully-hoist
```

2. Place config.js file in the /src directory

## Development Server

Start the development server on http://localhost:3001

```bash
npm start
```
